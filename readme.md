**Step one** 
Download source
https://github.com/israelomotayo/flask-practice

**Step two**

create venv
`python -m venv venv`

**Step three**

Activate venv windows: `venv\Scripts\activate`
Mac: `source venv/bin/activate`

**Step four**

Install requirements: `pip install -r requirements.txt`

**Step five**

run flask: `flask run`
